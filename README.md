# Calculator C++

Basic calculator developed in C++.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Console interaction

## Technologies

- C++
- Standard Library

## Installation

Clone the repository:

```bash
git clone https://github.com/KevinCarlotto/exo-calculator-c.git
```

Compile:

```bash
g++ main.cpp -o calculator
```

Run:

```bash
./calculator
```

Practice project focused on:

- C++ fundamentals
- Programming logic
- Functions
